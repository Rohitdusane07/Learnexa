# Learnexa - AI Study Companion

## Project Overview
Learnexa is an AI-powered study companion web platform designed to make learning smarter, faster, and personalized. The system helps students understand concepts, practice effectively, and track progress in one unified platform using Ollama AI API.

## Functional Requirements
1. **User Management**
   - Student signup and login.
   - Profile creation and customization.
   - Password encryption for security.

2. **Quiz Generation**
   - AI-generated MCQ quizzes based on selected topics via Ollama API.
   - Option to generate quizzes of different difficulty levels.

3. **Topic Analysis**
   - Personalized topic summaries.
   - Detailed insights into weak and strong areas.

4. **Study Roadmaps**
   - AI-driven personalized study plan based on performance.
   - Recommendations on which topics to prioritize.

5. **Progress Tracking**
   - Dashboard showing academic progress over time.
   - Analytics on weak areas and mastery levels.

6. **Question-Answer Assistant**
   - AI assistant using Ollama API to clarify doubts.
   - Instant responses for topic-specific questions.

7. **Interface & UX**
   - Clean, responsive web interface using HTML, CSS, and JS.
   - Session-based dashboard for ongoing study sessions.

## Non-Functional Requirements
- Fast response time (<2 seconds for AI responses).
- Scalable for multiple concurrent users.
- Secure data storage using MySQL.
- Cross-platform compatibility (mobile, tablet, desktop).
- Secure communication with Ollama API.

## Constraints
- Must integrate Ollama AI API for all AI functionalities.
- Must use PHP for backend logic and MySQL for database.
- Frontend should remain lightweight and user-friendly.
